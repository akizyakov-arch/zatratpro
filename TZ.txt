Общее ТЗ проекта
Название

MVP Telegram-бота для распознавания чеков, актов и накладных.

Цель

Создать минимально рабочий Telegram-бот, который:

принимает от пользователя фото документа,

отправляет изображение в OCR.Space,

получает распознанный текст,

показывает результат пользователю в Telegram,

затем передает OCR-текст в DeepSeek,

получает на выходе структурированный JSON,

показывает JSON пользователю в Telegram.

Ограничения MVP

На первом этапе не делать:

админку

роли

многопользовательские проекты

БД

веб-интерфейс

категории расходов

редактирование документов

подтверждение человеком

хранение истории

Docker на старте

деплой на сервер на старте

Среда разработки

Проект разрабатывается в WSL2 (Ubuntu), чтобы потом без боли:

завернуть в Docker

перенести на VPS

повторить локальную среду на сервере

Основной сценарий пользователя

Пользователь открывает Telegram-бота.

Отправляет фото чека, акта или накладной.

Бот принимает фото.

Бот отправляет изображение в OCR.Space.

Бот получает OCR-текст.

Бот показывает распознанный текст пользователю.

Бот отправляет OCR-текст в DeepSeek.

DeepSeek возвращает строго заданный JSON.

Бот показывает JSON пользователю.
